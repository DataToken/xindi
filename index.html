Xindi

Xindi is a lightweight CFML content management system ideally suited to smaller websites.

Download this project as a .zip file Download this project as a tar.gz file

Why choose Xindi?

Xindi is a small, lightweight and flexible CFML content management system. It's great for web developers looking for a simple way to make site content editable by their clients.

Open source, free and flexible, Xindi is a great choice for your next web project.

Why developers love Xindi:

  • open source and completely free
  • quick and easy set-up
  • back-end developed using CFML, FW/1, DI/1 and ValidateThis
  • front-end designed using HTML5, Twitter Bootstrap, jQuery and CKeditor
  • extensible and easy to customise
  • retain full markup control

Why clients love Xindi:

  • easy and fast content editing
  • secure control panel
  • WYSIWYG editor
  • upload images, documents, video and audio
  • search content with site search
  • search engine friendly
  • news feature with RSS feed
  • enquiry form
